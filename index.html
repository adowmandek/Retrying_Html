CSS BOX MODEL

Box model applies to block boxes, inline boxes only use some of the behavior defined in the box model. The model defines how the different parts of a box work together to create a box that you can see on the page.
Box Model properties

  • Border
  • Padding
  • Content
  • Margin
Total width = width + left padding + right padding + left border + right border + left margin + right margin
